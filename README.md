# New Fish Notifier

A simple quality-of-life mod for On-Together.

## Features
* Displays a notification in the chat whenever you catch a fish you haven't caught before.
* Tracks unique rarities! Catching a **Common Guppy** and then a **Rare Guppy** will count as two separate "New" discoveries.

## Installation
1.  Install via Thunderstore Mod Manager.
2.  Go fishing!
